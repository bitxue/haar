haar by Nathan Clack

nD haar transform and it's inverse.

This is a personal project wherein I play with the Haar transform. It's not particularly speedy, but it is implemented in C. Only float arrays for now.

API Reference

Dependencies

CMake (for the build)

Mylib (sort of optional)

googletest (optional)

Download

You can download this project in either zip or tar formats.
